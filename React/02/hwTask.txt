React компонент — це незалежна, перевикористовувана частина інтерфейсу користувача, яка описує вигляд і поведінку частини вашого веб-додатку або сторінки у вигляді JavaScript функції або класу. Компоненти в React можуть приймати дані, звані props, і повертати React елементи, які описують, що має бути відображено на екрані. Компоненти дозволяють розбити UI на незалежні, перевикористовувані частини, що полегшує їх управління та тестування.

Ось приклад простого React компоненту:
import React from 'react';

function Greeting() {

  return <h1>Hello!</h1>;

}

export default Greeting;
